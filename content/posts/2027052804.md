#### **左側：核心操作步驟**

**1. 安全註冊與身份驗證**
首先，通過專屬鏈接訪問 GMGN 官網。點擊註冊，建議使用專屬郵箱進行註冊，並設置包含大小寫字母、數字和特殊符號的高強度密碼。完成基礎註冊後，系統可能會提示進行基礎的 KYC（瞭解你的客戶）驗證，通常只需郵箱驗證即可開始使用大部分功能。這是保護你賬戶安全的第一道屏障。

**2. 熟悉核心數據看板**
登錄後，你將進入 GMGN 的主儀表板。重點關注以下幾個模塊：
*   **Smart Money 追蹤**：監控被標記為“聰明錢”的地址動向。
*   **鏈上大額異動**：實時捕捉巨額代幣轉賬，這往往是變盤的前兆。
*   **情緒指標**：綜合社交媒體熱度、搜索量等數據的市場情緒指數。
*   **代幣解鎖日程**：提前預知大量代幣釋放對市場的潛在衝擊。花時間逐一點擊查看，理解每個數據背後的意義。

**3. 申請並配置 API 金鑰**
這是與外部系統集成的核心。在賬戶設置中找到「API 管理」或「開發者中心」。
*   點擊「創建新的 API 金鑰」。
*   **權限設置是關鍵**：務必遵循 **“最小權限原則”**。如果僅用於讀取數據，就只勾選“讀取”權限，絕對不要勾選“交易”、“提現”等寫入權限。
*   創建後，系統會生成一對 **API Key** 和 **Secret Key**。**Secret Key 只會顯示一次**，必須立即安全地保存（如使用密碼管理器），之後再也無法查看。

**4. 測試 API 連接**
在獲得 API 金鑰後，建議先在 Postman 或類似的 API 測試工具中進行連接測試。使用官方文檔提供的基礎端點（例如，獲取 BTC 鏈上指標），輸入你的 API Key 進行調用。成功返回結構化數據（通常是 JSON 格式），即代表連接成功。這一步能排除基礎的配置錯誤。
